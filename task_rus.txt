Тестовое задание для Front-end разработчика.


Необходимо разработать SPA(single page application), которое выгружает данные из csv-файла и отображает их в виде таблицы.


В csv-файле находятся данные по пассажирам поезда (придумать данные). Заголовки таблицы находятся в первой строке файла.


Требования к таблице:
1. Таблица должна быть динамической. Кол-во колонок и их заголовки определяются данными из файла
2. Необходима возможность прямой или обратной сортировки данных по нажатию на заголовок колонки
3. При обновлении страницы, выбранная сортировка должна сохраниться
4. Подумать над оптимизацией отрисовки таблицы, если данных будет 100000 строк*


 Нельзя использовать готовые компоненты таблиц. Рекомендуется использовать React, Redux, Lodash, Bootstrap
